# Iot-svg-library

本仓库用于管理各类智慧医院、智慧建筑、物联网集成平台场景下的 **纯 SVG 矢量结构图与系统示意图**。  
所有图形均采用统一工程化风格，适用于 AIoT/IBMS 平台、可视化大屏、监控组态界面及技术文档。

---

## 📘 仓库目的

建立一个可长期维护、可扩展、可复用的 **IoT SVG 资产库**，用于展示各机电系统、医用设备系统、智慧化子系统的：

- 系统结构示意  
- 监控关系展示  
- 数据流向图  
- 参数监测点布局  
- 集成平台架构图

未来可作为：

- AIoT 平台前端组态图形库  
- IBMS 智慧医院/智慧楼宇界面资产库  
- 工程实施资料图形仓库  
- 文档/PPT/可视化展示图形源文件库

---

## 📁 文件命名规范（重要）

为保证可持续管理，本仓库所有 SVG 文件采用统一命名方式：
system-name_view-type_version.svg

命名字段说明：

| 字段 | 说明 |
|------|------|
| system-name | 子系统英文标识，全部小写、使用短横线分词 |
| view-type | 图形类型（overview / detail / diagram / piping 等） |
| version | v1 / v2 / v3，用于版本管理 |

---

## 🎨 SVG 风格设计规范（统一要求）

所有 SVG 均遵循以下标准（你之后每次请求我生成时也会自动遵循）：

- 纯 SVG 矢量，不包含位图元素  
- 扁平化工程图风格  
- 主色调：蓝色、灰色系  
- stroke-width 统一为 1.5  
- 常用基础图形：`rect`, `circle`, `line`, `polyline`, `path`  
- 监测点统一使用：小圆点 + 文本标注  
- 数据流使用蓝色箭头  
- 多子系统指向统一的 **智能化集成平台（AIoT/IBMS）**  
- 结构布局整洁，可视化逻辑清晰
